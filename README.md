# Practice9
Практика Лаба 9

Тема
Закріплення функціональних інтерфейсів, лямбд/method references та Java Time API на новому домені.

Активності на занятті
Заняття розпочинається з проходження тесту в системі Moodle за темою попереднього тижня (Collections, 15 хвилин). Основна частина заняття присвячена послідовному виконанню серії коротких практичних завдань із поступовим ускладненням. Ви працюватимете індивідуально у своєму середовищі розробки. Викладач супроводжуватиме процес, надаватиме допомогу та перевірятиме ваш код безпосередньо під час виконання завдань. Фінальним етапом стане фіксація результатів за допомогою Git.

Завдання
Завдання 1: Event Schedule (15 хв)

Мета: створити модель події та підготувати основу для FI та Java Time.

Створіть клас Event з полями: title, start, durationMinutes, zone, track.
Додайте метод end(), який повертає час завершення події.
Додайте метод label() (назва + трек).
Створіть 3-4 події у Main.
Завдання 2: Functional Interfaces (15 хв)

Мета: використати FI для задачі розкладу.

Реалізуйте EventLab.pick(List<Event>, Predicate<Event>).
Реалізуйте EventLab.labels(List<Event>, Function<Event, String>).
Реалізуйте EventLab.notifyAll(List<Event>, Consumer<Event>).
Реалізуйте EventLab.create(Supplier<Event>).
Створіть Predicate для ранкових подій (до 12:00).
Зберіть фільтр як композицію предикатів: використайте and/or/negate (мінімум 2 правила).
Додайте EventLab.findConflicts(List<Event>), який знаходить пари подій, що перетинаються у часі (через start/end і ZoneId).
Завдання 3: Лямбди та method references (15 хв)

Мета: навчитися переходити від анонімного класу до лямбди та method reference.

Створіть LambdaRefactorLab.sortAnonymous(...).
Перепишіть сортування на лямбду (sortLambda).
Перепишіть на method reference (sortMethodRef).
Завдання 4: Java Time (10 хв)

Мета: застосувати LocalDateTime/Instant/ZoneId на новому сценарії.

Реалізуйте DateTimeLab.toInstant(Event e) з урахуванням ZoneId.
Реалізуйте DateTimeLab.minutesBetween(Event a, Event b) через Duration.
Реалізуйте DateTimeLab.startInZone(Event e, String zone).
Завдання 5: Git-цикл (10 хв)

Мета: зафіксувати зміни через Git.

Створіть гілку feature/event-lambda-lab.
Зробіть 1-2 осмислені коміти.
За можливості - git push -u origin feature/event-lambda-lab.
Оцінювання
Оцінка за практичне заняття формується з двох складових:

Тестування: Результат проходження автоматичного тесту в Moodle (за темою Collections).
Практична робота: Викладач перевіряє виконання 5-ти завдань безпосередньо на занятті (фіксується у чек-листі):
Створення базової моделі (Завдання 1).
Правильне використання функціональних інтерфейсів (Завдання 2).
Успішний рефакторинг коду за допомогою лямбд та method references (Завдання 3).
Коректне застосування Java Time API (Завдання 4).
